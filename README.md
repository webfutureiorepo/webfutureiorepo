## 🚀 Getting Started

### Prerequisites
- Node.js >= 18
- npm / yarn / pnpm

### Installation
```bash
git clone https://github.com/Webfuture/webfutureiorepo.git
cd webfutureiorepo
npm install
```

### Run Locally
```bash
npm run dev
```

### Build for Production
```bash
npm run build
npm run start
```
